Since we are interested in the cloze probability in each condition, we need to calculate that, too. For this, well create a new dataframe called grouped_data because crucially, we are grouping all of our data by Item and Condition. Notably, this means we wont have random effects by Subject, only by Item. grouped_data has the following columns Item: the item number cp1: the encoded clause placement level l1: the encoded location level Cloze Prob: the cloze probability of the target word for each item and condition

mod <- aov(ClozeProb ~ cp1 * l1,
  data = grouped_data
)
shapiro.test(mod$residuals)

Our data is not normally distributed per the Shapiro-Wild test, barring us from using LM or ANOVA.

lm_model1 <- lmer(ClozeProb ~ 1 + cp1 * l1 +
